< a >
외부 문서를 연결시키는 태그.
주로 웹 페이지나 외부 사이트를 연결할 때 사용.
외부페이지로 이동하는 것 뿐만 아니라 내부의 특정 요소로 초점을 이동시킬 수도 있다. : href = #(id) 형식

href(hyper reference) : 필수 속성. 링크의 목적지가 되는 URL이 들어간다.
target : 선택 속성. 링크된 리소스를 어디에 표현할지 나타낸다. 
- _self : 우리가 보고 있던 화면 자체를 링크 페이지로 이동한다. (default) 
- _blank : 새로운 창, 빈 창에 외부 페이지가 나타난다.

ex) <a href="http://www.naver.com/" target="_blank">네이버</a>

< body >
본문 내용과 관련된 태그.

< br >
글의 줄바꿈을 생성.

ex) <br>

<form>
폼 데이터를 그룹화하여 실제 서버로 전송해주는 역할.

action : 폼 데이터를 처리하기 위한 서버의 주소.
method : 데이터를 전송하는 방식을 지정.
- get : URL을 통해서 데이터를 전달. (default)
- post : 숨겨서 데이터를 전달.

ex)
<form action=""http://localhost/login.php" method="post">
    <p> 아이디 : <input type="text" name="id" value="default value"> </p>
    <p> 비밀번호 : <input type="password" name="pwd" value="default value"> </p>
    <p> 주소 : <input type="text" name="address"> </p>
    <input type="submit"> 
    <p>textarea :
        <textarea cols="30" rows="3">default value</textarea>
    </p>
</form> 

< head >
본문 내용과 관련되지 않는 태그.

< hn >
문서 내의 제목을 표현할 때 사용.
제목의 레벨에 따라서 <h1>~<h6>까지 존재.
보통 <h1>은 페이지를 대표하는 큰 제목으로 주로 사용되며, 숫자가 올라갈수록 더 낮은 수준의 소제목을 나타낸다.

ex) <h1>역사</h1>

< img >
문서에 이미지 삽입.

src : 이미지 경로 지정.
alt : 이미지의 대체 텍스트 입력 = 서버에서 이미지를 못 가져오는 경우 엑스박스 표시 옆에 노출되는 문구.
title : 사진에 커서를 올렸을 때 나오는 도움말.
width / height : 이미지의 크기(가로, 세로) 지정.

★ width 나 height 둘 중 하나의 값만 조정하면 나머지 하나도 원본 비율에 맞춰 조정된다. 즉, 사진의 비율 유지
★ 두 속성 모두 변경할 경우 이미지의 비율이 왜곡된 채로 원하는 크기로 변경된다.

ex) <img src="/examples/images/img_monalisa.png" alt="모나리자" width="100" height="30">

<input type="text">
문장 입력 양식. 단순한 텍스트를 입력할 때 사용.

name : 서버로 보낼 때 지정되는 데이터의 이름.
value : 기본값.
placeholder : 사용자가 값을 입력하기 전에 미리 입력되어 있는 문장으로 입력 시작시 사라짐.

ex)
<input type="text" name="id" value="default value"> 

<input type="password">
암호를 입력하는 양식. 값이 노출되지 않음.

name : 서버로 보낼 때 지정되는 데이터의 이름.
value : 기본값.
placeholder : 사용자가 값을 입력하기 전에 미리 입력되어 있는 문장으로 입력 시작시 사라짐.

ex)
<input type="password" name="pwd" placeholder="암호 입력">

<input type="submit">
form의 값을 전송하는 버튼. 

value : 버튼의 이름 변경.

ex)
<input type="submit" value="전송"> 

< ol >
순서가 있는 리스트.

ex) 
<ol> 
    <li>콩나물</li> 
    <li>파</li> 
    <li>국간장</li> 
</ol>

< p >
글의 단락을 나타낸다.

ex) <p> HTML 최초의 일반 공개 설명은.. </p>

< strong >
글자를 굵게 표현.

ex) <strong>힘</strong> 

< table >
표.

border : 각 데이터의 테두리 두께.
colspan : 열 병합.
rowspan : 행 병합.

ex)
<table border="2">
    <thead>
        <tr>
            <th>이름</th> <th>성별</th> <th>주소</th> <th>회비</th>
        </tr>
    </thead>
    <tbody>
        <tr>         
            <td>최진혁</td> <td>남</td> <td rowspan="2">청주</td> <td>500</td>
        </tr>
        <tr>         
            <td>최유빈</td> <td>여</td> <td>1000</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>         
            <td colspan="3">합계</td> <td>1500</td>
        </tr>
    </tfoot>
</table>

< textarea >
여러 줄의 텍스트를 입력할 때 사용.

placeholder : 사용자가 값을 입력하기 전에 미리 입력되어 있는 문장으로 입력 시작시 사라짐.
rows : 행 길이를 조절
cols : 열 길이를 조절
태그 사이의 문장이 기본값으로 적혀 있음.

ex)
<textarea cols="30" rows="3">자기소개를 하세요.</textarea>

< title >
웹 페이지의 제목.

ex) <title>제목</title>

< ul >
순서가 없는 리스트.

ex)
<ul> 
    <li>콩나물</li> 
    <li>파</li> 
    <li>국간장</li> 
</ul>
